Module report 

Source
Expand description

Report generation: a spreadsheet-friendly CSV, and a re-importable JSON that round-trips the full FolderReport — every field the app computed, nested detections included — so a saved analysis can be dropped back onto the window later and rendered without re-decoding a single audio file.
