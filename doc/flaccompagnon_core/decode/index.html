Module decode 

Source
Expand description

Multi-format decoding via Symphonia, streamed into a StreamAnalyzer.

Every packet is exposed as a normalized f32 interleaved buffer used for spectral / clipping / stereo analysis. For integer PCM sources the exact integer sample values are reconstructed from those floats (the conversion is lossless for <= 24-bit content, since an f32 mantissa represents every integer up to 2^24 exactly) and fed to the bit-depth estimator.

decode_and_analyze_dsd
Decode a DSD file (DSF/DFF) through ffmpeg and stream the PCM into the analyzer. ffmpeg converts DSD to float PCM at dsd_rate / 8 (352.8 kHz for DSD64); the returned outcome carries that decoded rate, which is what the spectral analysis must use. No shell is involved; arguments go through Command::arg.
