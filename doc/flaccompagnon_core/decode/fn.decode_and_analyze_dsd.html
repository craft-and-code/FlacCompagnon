Expand description

Decode a DSD file (DSF/DFF) through ffmpeg and stream the PCM into the analyzer. ffmpeg converts DSD to float PCM at dsd_rate / 8 (352.8 kHz for DSD64); the returned outcome carries that decoded rate, which is what the spectral analysis must use. No shell is involved; arguments go through Command::arg.
