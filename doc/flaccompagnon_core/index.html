FlacCompagnon core analysis library.

This crate is intentionally free of any Tauri / UI dependency so that every analysis routine can be unit-tested in isolation with plain cargo test.

Public entry points:

  • analyze_file — analyze a single audio file.
  • analyze_folder — analyze every supported audio file inside a folder (non-recursive by default; see ScanOptions).
